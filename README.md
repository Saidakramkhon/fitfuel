# FitFuel

FitFuel is a simple web application created as a university project.  
The idea of the project is to provide healthy fitness meals and supplements through an online shop.

Users can register, log in, browse products, filter categories, add meals to cart, and place orders.  
Administrators can manage products by adding, editing, and deleting items.

## Main Features

- User registration and login
- Session-based authentication
- Product listing with images
- Search and category filters
- Shopping cart
- Checkout page
- Admin panel for product management
- Product categories such as Bulk, Cut, Snack, Supplement, Vegan

## Technologies Used

- PHP
- MySQL
- JavaScript
- HTML
- CSS
- XAMPP (Apache + MySQL)

## Project Structure

- `frontend/` → user interface files
- `backend/` → API and server-side logic
- `database.sql` → database structure and sample data

## How to Run the Project

1. Install XAMPP
2. Move the project folder into `htdocs`
3. Import `database.sql` into phpMyAdmin
4. Start Apache and MySQL
